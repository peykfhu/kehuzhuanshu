使用政策Acceptable Use Policy

最近更新：2026 年 4 月 18 日Last updated: April 18, 2026

1. 概述

本使用政策（Acceptable Use Policy）规定了 AnyHop AI 平台服务的合理使用规范。所有用户在使用本平台时必须遵守本政策。违反本政策可能导致账户被暂停或永久终止。

2. 允许的用途

您可以将本平台用于以下合法目的：

  • 软件开发与编程辅助
  • 内容创作与文案撰写
  • 数据分析与研究
  • 教育和学习
  • 客户服务与商业运营
  • 合法的商业应用开发

3. 禁止行为

以下行为严格禁止，违反者将被立即封停账户，且不予退还余额。

3.1 违法活动

  • 生成、传播违反法律法规的内容
  • 利用 AI 进行诈骗或其他犯罪活动
  • 侵犯他人知识产权或隐私
  • 制作或传播色情、暴力或恐怖相关内容

3.2 滥用行为

  • 对平台进行恶意攻击（DDoS、暴力破解等）
  • 尝试绕过安全限制或速率限制
  • 利用漏洞获取未授权的访问权限
  • 批量注册账户或使用自动化工具套取免费额度
  • 共享、转售或二次分发 API 访问权限

3.3 有害内容

  • 生成虚假信息或深度伪造内容用于欺骗
  • 利用 AI 制作垃圾邮件或骚扰信息
  • 生成歧视性、仇恨性言论
  • 制作可能危害公共安全的内容

4. 资源使用限制

  • 遵守平台设定的速率限制和并发限制。
  • 不得发送过大或明显浪费资源的请求。
  • 不得使用本平台进行加密货币挖矿或类似的资源密集型活动。

5. API Key 安全

  • API Key 仅限注册本人使用，不得与他人共享。
  • 不得在客户端代码、公开仓库或前端应用中暴露 API Key。
  • 发现 API Key 泄露时应立即重新生成。

6. 内容审核

我们保留对 API 使用情况进行监控和审核的权利。发现违反本政策的行为时，我们可能会：

  • 发出警告通知
  • 临时限制或暂停服务
  • 永久终止账户
  • 在必要时向有关部门举报

7. 举报机制

如果您发现有用户违反本使用政策，请通过控制台内的联系方式向我们举报。

8. 政策更新

我们保留随时修改本使用政策的权利。修改后的政策将在本页面公布。

1. Overview

This Acceptable Use Policy governs the proper use of AnyHop AI platform services. All users must comply with this policy. Violations may result in account suspension or permanent termination.

2. Permitted Uses

You may use the Platform for the following lawful purposes:

  • Software development and coding assistance
  • Content creation and copywriting
  • Data analysis and research
  • Education and learning
  • Customer service and business operations
  • Legitimate commercial application development

3. Prohibited Activities

The following activities are strictly prohibited. Violators will have their accounts immediately suspended with no balance refund.

3.1 Illegal Activities

  • Generating or distributing content that violates laws
  • Using AI for fraud or other criminal activities
  • Infringing on others' intellectual property or privacy
  • Creating or distributing pornographic, violent, or terrorist-related content

3.2 Abuse

  • Malicious attacks on the platform (DDoS, brute force, etc.)
  • Attempting to bypass security or rate limits
  • Exploiting vulnerabilities for unauthorized access
  • Mass account registration or automated free credit harvesting
  • Sharing, reselling, or redistributing API access

3.3 Harmful Content

  • Generating misinformation or deepfakes for deception
  • Creating spam or harassment using AI
  • Generating discriminatory or hateful speech
  • Creating content that may endanger public safety

4. Resource Limits

  • Comply with platform rate and concurrency limits.
  • Do not send excessively large or wasteful requests.
  • Do not use the platform for cryptocurrency mining or similar resource-intensive activities.

5. API Key Security

  • API Keys are for registered user's personal use only — do not share.
  • Do not expose API Keys in client code, public repos, or frontend apps.
  • Regenerate immediately if an API Key is compromised.

6. Content Moderation

We reserve the right to monitor and review API usage. Upon discovering policy violations, we may:

  • Issue a warning notice
  • Temporarily restrict or suspend service
  • Permanently terminate the account
  • Report to authorities when necessary

7. Reporting

If you discover a user violating this policy, please report it through the console contact channels.

8. Policy Updates

We reserve the right to modify this policy at any time. Updated policies will be posted on this page.
